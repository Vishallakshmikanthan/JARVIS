# JARVIS + FRIDAY Assistant (IronMan AI)

This is the repository for the IronMan AI project, a sophisticated assistant inspired by Tony Stark's systems.

## Project Structure

- `core/`: Core logic and orchestration.
- `voice/`: Voice recognition and text-to-speech.
- `auth/`: User authentication and security.
- `skills/`: Modular capabilities for the assistant.
- `gui/`: Graphic user interface.
- `models/`: Machine learning and NLP models.
- `assets/`: Static files and resources.
- `logs/`: Runtime logs for debugging.

## Setup

1. Create a virtual environment: `python -m venv venv`
2. Activate environment:
   - Windows: `.\venv\Scripts\activate`
   - Unix/macOS: `source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Set up `.env` with API keys and secrets.
5. Run: `python main.py`
